Installing JSmol/Jmol on Your Server

You may install JSmol/Jmol anywhere on your server that you are allowed to install files. Copy the jsmol.zip file from the Jmol directory (created when you unzipped the Jmol download) to the directory on the server where you want the files. Unzip jsmol.zip. This will create a directory 'jsmol' in which all the necessary files are stored. Make sure you know where you are installing the jsmol directory as you will have to specify the path to Jmol in web pages you create.
