Testing Web Pages Created by Jmol Web Page Maker on Your Local Machine

(Because of security constraints only Mozilla browsers seem to work for local testing as of Jan. 2013)

The javascript built into the pages checks to see whether the page is being loaded from a server or the local machine.  If it is being loaded from your local machine the path to Jmol used is the relative path specified in the local path text box. The directory containing the page to be tested must be the same as the directory containing the 'jsmol' directory.

  1. Create your web page specifying the local path as ".", without the quotes.
  2. Open the .html file in the browser you want to use for testing.  The page should work just like it would on a server.
